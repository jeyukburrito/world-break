## Issue Diagnosis & Recommendation

### `npm run build` 실패 (Playwright 타입 누락)
- **현상**: `./~/gstack/browse/src/browser-manager.ts` 파일에서 `playwright` 모듈을 찾지 못해 타입 에러 발생.
- **원인**: `tsconfig.json`의 `"include": ["**/*.ts"]` 설정이 프로젝트 외부 경로인 `~/` 디렉토리까지 포함하여 타입 체크를 시도함.
- **해결 방안**: `tsconfig.json`의 `exclude` 목록에 `~/` 경로를 추가하여 타입 체크 대상에서 제외할 것을 권장함.
    ```json
    "exclude": ["node_modules", "**/~", "**/~/**"]
    ```
