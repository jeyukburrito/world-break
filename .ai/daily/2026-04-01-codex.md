# 2026-04-01-codex

## OAuth Callback Canonical Host Hotfix
- Added a callback-host normalization guard in `app/auth/callback/route.ts`.
- When `/auth/callback` is hit on a non-local non-canonical host, the route now issues a `307` redirect to `https://world-break.vercel.app/auth/callback` with the original query string preserved before exchanging the Supabase auth code.

## Middleware Canonical Host Hotfix
- Added a global host normalization guard in `middleware.ts`.
- When any app request arrives on a non-local non-canonical host, middleware now issues a `307` redirect to the same path/query on `https://world-break.vercel.app` before auth or guest routing runs.
