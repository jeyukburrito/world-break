# PRD — World Break

## 제품 개요

TCG(트레이딩 카드 게임) 플레이어를 위한 개인 전적 기록 모바일 PWA.
대회/비대회 경기 결과를 기록하고, 승률·덱 통계를 확인한다.

## 대상 사용자

TCG를 정기적으로 플레이하는 개인 유저 (한국 TCG 커뮤니티 중심).
여러 게임을 동시에 플레이하는 유저도 지원 (포켓몬, 유희왕, 원피스 등 22개 게임).

## 핵심 기능 (MVP)

| 기능 | 설명 |
|------|------|
| 매치 기록 | BO1/BO3 결과 입력, 선후공, 상대 덱, 날짜 기록 |
| 대회 세션 | 토너먼트 단위로 경기를 묶어 라운드별 기록 |
| 덱 관리 | 게임별 덱 목록, 마지막 사용 덱 프리필 |
| 대시보드 | 기간별·게임별·카테고리별 승률·통계 |
| 공유 | OG 이미지 기반 매치/대회 결과 SNS 공유 |
| PWA | 오프라인 지원, 홈화면 설치, 모바일 최적화 |
| 인증 | Google OAuth (Supabase Auth) |

## 범위 외 (v1)

- 멀티플레이어 / 소셜 기능 (팔로우, 리그 랭킹)
- 덱 리스트 공개/공유
- 상대방 정보 DB (상대 이름 검색 등)
- 공식 대회 API 연동

## 주요 UX 제약

- **모바일 퍼스트**: 한 손으로 빠르게 입력 가능해야 함
- **오프라인 입력**: 대회 중 인터넷 불안정 환경에서도 기록 가능 (PWA캐시)
- **빠른 입력 흐름**: 매치 결과 기록이 3탭 이내로 완결되어야 함
- **게스트 모드**: 로그인 없이 체험 가능 (쿠키 기반 임시 세션)

## 성공 지표

- 대회 당일 실제 전적 기록 사용률
- 매치 기록 완료율 (입력 시작 → 저장)
